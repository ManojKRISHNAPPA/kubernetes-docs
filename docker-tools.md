## 5. DevOps + MLOps Integration Flow

### DevOps Pipeline
- Code Commit → GitHub
- Build & Test → GitHub Actions / Jenkins
- Infrastructure Provision → Terraform
- Container Build → Docker
- Deployment → Amazon EKS
- Monitoring → CloudWatch / Prometheus

### MLOps Pipeline
- Data Ingestion → Amazon S3
- Model Training → SageMaker
- Experiment Tracking → MLflow
- Model Registry → SageMaker Registry
- Deployment → SageMaker Endpoint / EKS
- Monitoring → Model Monitor
